# Medical Report Classification Using NLP

This project classifies medical reports into specialties such as **Cardiology, Neurology, Oncology, and Orthopedics** using Natural Language Processing (NLP) techniques.

## 📂 Project Structure
- `src/` – Source code (preprocessing, training models)
- `data/` – Dataset and configuration files
- `notebooks/` – Jupyter notebooks for experiments
- `outputs/` – Model evaluation results (confusion matrix, accuracy charts)
- `main.py` – Entry point for running the project
- `requirements.txt` – Dependencies
